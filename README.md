# Hong-Cinema
영화 예매 및 정보 제공 사이트

## 1. 기술스택
- Tools : `ECLIPSE IDE, SQL DEVELOPER, Visual Studio Code`
- Language : `Java, JavaScript, HTML5, CSS3`
- Database : `Oracle`
- Framework : `Spring, MyBatis`

## 2. 기능
- 회원 CRUD
- 영화 예매, 결제
  - 영화 진흥 위원회 OPEN API 사용
- 영화 정보 조회
  - 네이버 영화 OPEN API 사용
- 게시판 CRUD
