## 1. 사용 기술 및 라이브러리
- `Java, JavaScript, HTML5, CSS3`
- `Spring MVC, Mybatis`
- `JQuery, I'mport;(JQuery 기반의 결제 연동 서비스 라이브러리)`
- `JSP, Apache Tomcat`
- `Oracle`

## 2. 기능
- 회원 CRUD
- 영화 예매, 결제
  - 영화 진흥 위원회 OPEN API 사용
- 영화 정보 조회(검색)
  - 네이버 영화 OPEN API 사용
- 게시판 CRUD
